## Option 1: GitHub + jsDelivr (Fastest & Free)

### Step 1: Create a GitHub Repository

```bash
# Navigate to your themes folder
cd /Users/saurabh/Documents/portfolio/public/themes

# Initialize git (if not already)
git init

# Add files
git add .
git commit -m "Initial commit: Solanke Themes v1.0.0"

# Create a new repository on GitHub (https://github.com/new)
# Then push your code
git remote add origin https://github.com/YOUR-USERNAME/solanke-themes.git
git branch -M main
git push -u origin main
```

### Step 2: Your CDN URL is Ready!

Once pushed to GitHub, your theme is instantly available via jsDelivr:

```html
<!-- Latest version -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@main/solanke-themes.css">

<!-- Specific version (recommended for production) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css">

<!-- Minified version (auto-generated by jsDelivr) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@main/solanke-themes.min.css">
```

### Step 3: Create Releases for Version Control

```bash
# Tag your version
git tag v1.0.0
git push origin v1.0.0

# Go to GitHub ‚Üí Releases ‚Üí Create New Release
# Tag: v1.0.0
# Title: Solanke Themes v1.0.0
# Description: Initial release with Luxury Green and Corporate Blue themes
```

**‚úÖ That's it! Your theme is now available globally via CDN!**

## Option 2: npm Package (Professional)

### Step 1: Setup npm Package

```bash
cd /Users/saurabh/Documents/portfolio/public/themes

# Login to npm (create account at https://www.npmjs.com if needed)
npm login

# Publish your package
npm publish
```

### Step 2: Your npm Package is Live!

Users can now install your theme:

```bash
npm install solanke-themes
```

And use it:

```javascript
// In your CSS/SCSS
@import 'solanke-themes';

// Or in your JS
import 'solanke-themes/solanke-themes.css';
```

### Step 3: CDN via unpkg

Once published to npm, it's automatically available via unpkg:

```html
<link rel="stylesheet" href="https://unpkg.com/solanke-themes@1.0.0/solanke-themes.css">
```

### Updating Your Package

```bash
# Update version in package.json
# Then publish
npm version patch  # 1.0.0 ‚Üí 1.0.1
npm publish

npm version minor  # 1.0.1 ‚Üí 1.1.0
npm publish

npm version major  # 1.1.0 ‚Üí 2.0.0
npm publish
```

## Option 3: GitHub Pages (Custom Domain)

### Step 1: Enable GitHub Pages

1. Go to your GitHub repository
2. Settings ‚Üí Pages
3. Source: Deploy from branch
4. Branch: `main` ‚Üí `/docs` or root
5. Save

### Step 2: Your Theme URL

```html
<link rel="stylesheet" href="https://YOUR-USERNAME.github.io/solanke-themes/solanke-themes.css">

<!-- Or with custom domain -->
<link rel="stylesheet" href="https://themes.yourdomain.com/solanke-themes.css">
```

## üéØ Using Your Theme in Projects

### Example 1: Simple HTML Project

```html
<!DOCTYPE html>
<html lang="en" data-color-theme="luxury">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Project</title>
  
  <!-- Your Published Theme -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css">
</head>
<body data-theme-enabled>
  <div class="glass-theme" style="padding: 2rem; margin: 2rem;">
    <h1 class="text-gradient-theme">Hello World!</h1>
    <button class="btn-theme-primary">Get Started</button>
  </div>
</body>
</html>
```

### Example 2: React Project

```bash
npm install solanke-themes
```

```javascript
// App.js or index.js
import 'solanke-themes/solanke-themes.css';
import { useEffect } from 'react';

function App() {
  useEffect(() => {
    // Set default theme
    document.documentElement.setAttribute('data-color-theme', 'luxury');
    document.body.setAttribute('data-theme-enabled', '');
  }, []);

  return (
    <div className="container">
      <div className="glass-theme" style={{ padding: '2rem', margin: '2rem' }}>
        <h1 className="text-gradient-theme">My React App</h1>
        <button className="btn-theme-primary">Click Me</button>
      </div>
    </div>
  );
}
```

### Example 3: Next.js Project

## üîí Best Practices

### Version Pinning (Recommended)

```html
<!-- ‚úÖ GOOD: Pinned version -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css">

<!-- ‚ùå BAD: Always uses latest (can break your site) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@main/solanke-themes.css">
```

### Integrity Hash (Extra Security)

```bash
# Generate SRI hash
cat solanke-themes.css | openssl dgst -sha384 -binary | openssl base64 -A
```

```html
<link rel="stylesheet" 
      href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css"
      integrity="sha384-YOUR-HASH-HERE"
      crossorigin="anonymous">
```

### Preload for Performance

```html
<link rel="preload" 
      href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css"
      as="style">
<link rel="stylesheet" 
      href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css">
```

## üìù Example: Complete Integration

```html
<!DOCTYPE html>
<html lang="en" data-color-theme="luxury">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Portfolio</title>
  
  <!-- Preload theme for better performance -->
  <link rel="preload" 
        href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css"
        as="style">
  
  <!-- Load theme -->
  <link rel="stylesheet" 
        href="https://cdn.jsdelivr.net/gh/YOUR-USERNAME/solanke-themes@v1.0.0/solanke-themes.css"
        crossorigin="anonymous">
</head>
<body data-theme-enabled>
  
  <!-- Your content using theme classes -->
  <nav class="glass-theme" style="padding: 1rem;">
    <button class="btn-theme-primary">Home</button>
    <button class="btn-theme-secondary">About</button>
  </nav>

  <main class="container">
    <div class="card-theme hover-lift">
      <h1 class="text-gradient-theme">Welcome</h1>
      <p class="text-theme-dark">Beautiful themed content</p>
    </div>
  </main>

  <script>
    // Theme persistence
    const savedTheme = localStorage.getItem('color-theme') || 'luxury';
    const isDark = localStorage.getItem('dark-mode') === 'true';
    
    document.documentElement.setAttribute('data-color-theme', savedTheme);
    if (isDark) document.documentElement.classList.add('dark');

    // Theme switcher
    function switchTheme(theme) {
      document.documentElement.setAttribute('data-color-theme', theme);
      localStorage.setItem('color-theme', theme);
    }
  </script>
</body>
</html>
```
